# תיק עבודות - ליעד שאבי

פרויקט פורטפוליו אישי נקי, מונגש ורספונסיבי המותאם להתקנה כאפליקציה (PWA).

> [!NOTE]
> בהתאם להנחיות, הוסרו משאבים שאינם קשורים ישירות לפיתוח האתר עצמו (כמו קובצי סיכום עבודה, walkthrough ומשאבים נלווים אחרים) במטרה לחסוך בזמן וצריכת טוקנים, ולשמור על פרויקט ממוקד ויעיל.
> אני אשתדל כעת לבצע את כל הבקשות הבאות בצורה ממוקדת ומהירה ככל הניתן כדי לחסוך לך זמן!

## מבנה הפרויקט
- `index.html` - הקובץ המרכזי המכיל את שלד ה-HTML, עיצוב ה-CSS וקוד ה-JavaScript יחד במקום אחד.
- `sw.js` - Service Worker המאפשר עבודה במצב לא מקוון.
- `manifest.json` - קובץ הגדרות ה-PWA.
- `images/` - תמונות ומוקאפים של הפרויקטים והאייקונים.
- `docs/` - מסמכים רלוונטיים (כגון מסמכי אפיון).
